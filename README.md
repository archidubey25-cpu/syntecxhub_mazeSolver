# Maze Solver using A* Search

## 🧩 Project Description
This project implements the A* search algorithm to find the shortest path in a maze.
- 0 = free space
- 1 = wall  
It calculates the optimal route from a start point to a goal point using a heuristic (Manhattan distance) and visualizes the path using matplotlib.

---

## ⚙️ Steps to Run
1. Install Python 3.14 or higher.
   
2. Install required package:
   ```bash
   pip install matplotlib

3. Run the program:
   ```bash
   python maze_solver.py
